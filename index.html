patch|waste

Detect patch-size regressions from SteamPipe preview build output. Rust CLI with CI-friendly exit codes and baseline budget gating.

Install
cargo install --git https://github.com/patchwaste/patchwaste patchwaste
$ patchwaste analyze --input BuildOutput --out patchwaste-out
